# Svelonia.Data

`Svelonia.Data` implements the Mediator pattern to help you build clean, decoupled architectures. It separates the code that sends a request (UI) from the code that processes it (Logic).

## Key Components

*   **[Mediator Pattern](./Mediator.md)**
    Using `IRequest`, `ICommand`, and `IMediator` to dispatch messages.
