# Svelonia.Core

The core library for Svelonia, providing the fundamental building blocks for reactive UI development.

## Features

*   **Fine-Grained Reactivity**: `State<T>`, `Computed<T>`, and dependency tracking.
*   **Component System**: Base `Component` class with lifecycle management.
*   **Functional Controls**: `IfControl`, `AwaitControl` for declarative logic.
*   **Animation**: Simple transition primitives.
