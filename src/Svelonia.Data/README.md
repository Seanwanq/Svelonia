# Svelonia.Data

A data handling library implementing the Mediator pattern, designed for clean architecture and AOT compatibility.

## Features

*   **Mediator Pattern**: `IRequest`, `ICommand`, and `IRequestHandler`.
*   **AOT Support**: Source generators create static registries to avoid runtime reflection.
*   **Dependency Injection**: Integrates with `Microsoft.Extensions.DependencyInjection`.
