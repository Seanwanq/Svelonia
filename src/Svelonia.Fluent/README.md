# Svelonia.Fluent

A library that wraps Avalonia UI controls with a fluent, chainable API, enabling a declarative C# developer experience.

## Features

*   **Fluent API**: Chain methods like `.Text("Hello").FontSize(20)` instead of object initializers.
*   **State Binding**: Seamlessly bind properties to `State<T>` values.
*   **Styling Utilities**: Tailwind-like color palettes (`Tw`) and layout helpers.
*   **Responsive Design**: Built-in media queries and platform detection.
