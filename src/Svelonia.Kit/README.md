# Svelonia.Kit

The application kit for Svelonia, providing the high-level structure for building single-page applications (SPAs).

## Features

*   **Routing**: URL-based navigation with parameters and guards.
*   **Page System**: `Page` class with async loading lifecycle (`OnLoadAsync`).
*   **NavigationHost**: The control that renders the current route.
*   **Layouts**: Support for nested layouts.
