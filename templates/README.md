# Svelonia Templates

This package contains project templates for the Svelonia framework.

## Included Templates

- `svelonia.app`: A standard Svelonia application template with basic routing and state management examples.

## Usage

Install the templates via dotnet CLI:

```bash
dotnet new install Svelonia.Templates
```

Create a new project:

```bash
dotnet new svelonia.app -n MyAwesomeApp
```
